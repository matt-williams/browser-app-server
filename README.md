# browser-app-server

Can you build a SIP application server, with media processing, in a web browser? Yes, you can.

Built for TADHack-mini London, 9-10 April 2016.

Using:
- WebRTC
- SIP.js
- clmtrackr
- Recorderjs
- Three.js
- MakeHuman
